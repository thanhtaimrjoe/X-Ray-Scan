# Vector Item Assets

This folder contains the first Codex-authored item asset source set for X-Ray Scan.

Generated by:

```text
tools/generate_item_vector_assets.py
```

Files:

- `item_*.svg`: editable vector source/output for each MVP item.
- `item_vector_preview_sheet.svg`: SVG review sheet.
- `item_vector_preview_sheet.png`: PNG review sheet for quick visual approval.

Current status:

- Candidate source set for review.
- Not yet promoted into `app/assets/images/items/`.
- Not yet wired into Flame gameplay rendering.

Workflow:

1. Edit `tools/generate_item_vector_assets.py`.
2. Run `python tools/generate_item_vector_assets.py`.
3. Review `item_vector_preview_sheet.png`.
4. After approval, wire the generated shapes or exported PNGs into the app.
